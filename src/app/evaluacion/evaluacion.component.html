Formula usada para calcular macros

1) Calorias: TMB Harris-Benedict: 88.362 + (13.397 x peso kg) + (4.799 x altura cm) - (5.677 x edad). Luego se multiplica por actividad (sedentario 1.2, ligero 1.375, moderado 1.55, intenso 1.725) y se ajusta por objetivo (-500 perder, +500 ganar).

2) Proteina: base proteica = masa magra (si existe) o (peso - masa grasa). Factor por objetivo: 1.6 g/kg perder, 1.4 g/kg mantener, 1.8 g/kg ganar.

3) Grasa y carbohidrato: grasa = 35% de las calorias restantes tras proteina, y carbohidratos = calorias restantes finales / 4.

Aplicado a este paciente:

TMB = 88.362 + (13.397 x {{ calc.peso | number:'1.0-1' }}) + (4.799 x {{ calc.altura | number:'1.0-1' }}) - (5.677 x {{ calc.edad }}) = {{ calc.tmb | number:'1.0-0' }} kcal.

Calorias objetivo = {{ calc.tmb | number:'1.0-0' }} x {{ calc.factorActividad }} ({{ calc.actividadLabel }}) {{ calc.ajusteObjetivo >= 0 ? '+' : '-' }} {{ Math.abs(calc.ajusteObjetivo) }} = {{ calc.caloriasObjetivo | number:'1.0-0' }} kcal.

Proteina = {{ calc.proteinaBase | number:'1.0-1' }} kg ({{ calc.proteinaBaseOrigen }}) x {{ calc.proteinasPorKg }} = {{ calc.proteinasDia }} g/dia.

Grasa = ({{ calc.caloriasPostProteina | number:'1.0-0' }} x 0.35) / 9 = {{ calc.grasasDia }} g/dia.

Carbohidratos = ({{ calc.caloriasObjetivo | number:'1.0-0' }} - {{ calc.proteinasDia }}x4 - {{ calc.grasasDia }}x9) / 4 = {{ calc.carbohidratosDia }} g/dia.
